Place trip photos directly in this folder.

Filename format:
<city-slug>-<anything>.<ext>

Examples:
src/assets/trips/tokyo-1.jpg
src/assets/trips/tokyo-night-market.png
src/assets/trips/grand-cayman-island-beach.webp

Supported city slugs:
orlando, new-orleans, los-angeles, denver, dallas, new-york, chicago,
grand-cayman-island, toronto, london, munich, budapest, krakow, tokyo, kyoto-osaka,
brisbane, melbourne, sydney
